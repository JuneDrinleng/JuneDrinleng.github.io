## 1 Getting to Know Arrays

For some application scenarios, such as when we need to calculate an average while also identifying which numbers are greater than the average, we need to record the input numbers. In such cases, defining many variables is often impractical (a brute-force approach). How to address this situation is where arrays come into play.

~~~java
int[] numbers = new int[100]
~~~

Create an array named `numbers` that can hold 100 `int` values.

Specifically, filling in numbers:

~~~
numbers[index] = values_index
~~~

For example:

## 2 Creating Arrays

An array is a container with the characteristic that its size cannot be changed once created; all elements have the same type.

For example, in section 1, the `numbers` array:

~~~java
int[] numbers = new int[100]
~~~

After creation, it can only store `int` values and hold exactly 100 elements.

The creation format is as follows:

~~~java
<type>[]<arrayName> = new <elementType>[elementCount]
~~~

Note that array indices start from 0.

## 3 Array Elements

Accessing array elements is done via indices. For instance, to retrieve the fifth element of the array `numbers`:

~~~java
numbers[4]
~~~

Assigning values also works:

~~~java
numbers[4] = 11
~~~

Note that even if your index exceeds the array's boundary, the compiler won't detect it during code writing; you only find out during runtime that the index is out of bounds.

Thus, setting an array size based on user input without knowing the number of inputs is risky. We should set the array size according to the number of inputs read from the user to avoid index overflow issues.

Arrays have an inherent property called `length`:

~~~java
for (int i = 0; i < grade.length; i++) {
    operation
}
~~~

Here, `grade` is the array.

## 4 Array Variables

Besides initializing and assigning values step by step (using `new`), you can also assign values directly during initialization:

~~~java
int[] scores = {97, 87, 97, 88, 67, 22}
~~~

![image-20250321204025975](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250321204025975.png)

Note that ordinary variables correspond to their values, but for arrays, the array and data are merely pointers:

![image-20250321204144294](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250321204144294.png)

Thus, when `int[] b = a`, `b` also points to the same memory location as `a`:

![image-20250321204231282](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250321204231282.png)

Modifying the first element via `b` will also change the first element of `a` through its index.

Summary:

![image-20250321204353164](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250321204353164.png)

If two arrays have different indices, their memory addresses differ, even if their contents are the same, they are still distinct arrays:

![image-20250321204522636](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250321204522636.png)

Therefore, comparing whether two arrays are equal cannot be done directly with `==`; instead, you must compare each element through a loop to determine if they are equal arrays.

## 5 Task: Vote Statistics
~~~java
import java.util.Scanner;

public class VoteStatistic {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int x;
        int[] numbers = new int[10];
        x = in.nextInt();
        while(x != -1) {
            if(x >= 0 && x <= 9) {
                numbers[x]++;
            }
            x = in.nextInt();
        }
        for(int i = 0; i < numbers.length; i++) {
            System.out.println(i + " : " + numbers[i]);
        }
    }
}

~~~

Directly create an array counting the occurrences of inputs [0,9]

## 6 Traversing Arrays

![image-20250321205734407](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250321205734407.png)

Generally, a for loop starting from 0 to length is used for traversal

Other options include:

~~~java
for (int k : data) {
    operation
}
~~~

This is the for-each loop, where k represents each element in data. It's particularly suitable for traversing every element in an array, but its drawback is that it doesn't know the index position. Note that:

~~~java
for (int k : data) {
    k = 9;
}
~~~

This cannot change the array's content

This essentially checks if the next prime is a prime by dividing it with primes in the prime table

Other methods also exist:

~~~java
boolean[] isprime = new boolean[100];
for (int i = 0; i < isprime.length; i++) {
    isprime[i] = true;
}
for (int i = 2; i < isprime.length; i++) {
    if (isprime[i]) {
        for (int k = 2; i * k < isprime.length; k++) {
            isprime[i * k] = false;
        }
    }
}
for (int i = 2; i < isprime.length; i++) {
    if (isprime[i]) {
        System.out.println(i);
    }
}
~~~

This method marks non-primes as multiples of primes, leaving only primes

## 8 Two-Dimensional Arrays

~~~java
int[][] a = new int[3][5];
~~~

Constructs a 3x5 matrix-like array

Two-dimensional arrays can also be initialized like one-dimensional arrays:

~~~java
int[][] a = {
    {1, 2, 3, 4},
    {1, 2, 4},
};
~~~

Constructs a 2x4 array, with missing elements automatically filled with zeros

A complex code example for a 3x3 game board:
~~~java
import java.util.Scanner;

public class board {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        final int Size = 3;
        int[][] board = new int[Size][Size];
        for (int i = 0; i < Size; i++) {
            for (int j = 0; j < Size; j++) {
                System.out.print("Enter the value of board line " + (i + 1) + " and column " + (j + 1) + ":");
                board[i][j] = in.nextInt();
            }
        }
        MainLOOP:
        for (int i = 0; i < Size; i++) {
            int const_num = board[i][0];
            for (int j = 0; j < Size; j++) {
                if (board[i][j] != const_num) {
                    break;
                } else if (board[i][j] == const_num && j == Size - 1) {
                    System.out.print(const_num + " wins ");
                    break MainLOOP;
                }
            }
        }
        MainLOOP2:
        for (int j = 0; j < Size; j++) {
            int const_num = board[0][j];
            for (int i = 0; i < Size; i++) {
                if (board[i][j] != const_num) {
                    break;
                } else if (board[i][j] == const_num && i == Size - 1) {
                    System.out.print(const_num + " wins ");
                    break MainLOOP2;
                }
            }
        }
        int const_num = board[0][0];
        for (int i = 0; i < Size; i++) {
            if (board[i][i] != const_num) {
                break;
            } else if (board[i][i] == const_num && i == Size - 1) {
                System.out.print(const_num + " wins ");
                break;
            }
        }
        const_num = board[0][Size - 1];
        for (int i = 0; i < Size; i++) {
            if (board[i][Size - 1 - i] != const_num) {
                break;
            } else if (board[i][Size - 1 - i] == const_num && i == Size - 1) {
                System.out.print(const_num + " wins ");
                break;
            }
        }
    }
}

~~~
