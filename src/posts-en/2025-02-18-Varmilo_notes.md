## 0 Background
The author purchased a red-axis mechanical keyboard from Amilo Hua Danniang in 2024. When I purchased it, I was impressed by the excellent appearance of the keyboard and the familiar red axis. However, in the past year of use, I have encountered many problems, including but not limited to the win key suddenly not responding (the win key of the notebook still remains It can be used), the entire mechanical keyboard is locked, and I don’t know where to play/pause/cut songs, increase/decrease volume, mirror sound, what the four buttons in the upper right corner correspond to, etc. After browsing the Internet, some of these problems were not solved, so I specifically asked customer service today to solve them.  
In order to avoid forgetting it later and to make it easier for netizens to use, I wrote a note to record the use of this keyboard.

## 1 Keyboard Lock
Unlike Cherry's red-axis mx250 keyboard and other keyboards, Amilo's keyboard volume key adjustment is not on `Fn`+`F2`. Therefore, when the author tried `Fn`+`F2` several times, the result was that the mechanical keyboard was directly locked and unusable.  

When the keyboard is locked, the first of the three indicators in the upper left corner will go out. To solve this problem, just hold down `Fn`+`Del`.

## 2 The win key fails

As a heavy win key user, the failure of the win key has brought great inconvenience to me (someone has no apps on the desktop, and all the apps are included in the win key menu). The method to trigger the failure of the win key (or lock the win key) is very simple. Just hold down `Fn` and the win key on its right:

![IMG_20250218_150028](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/IMG_20250218_150028.jpg)

Of course, under normal circumstances, the trigger is very accidental, so to unlock it, you only need to press the two keys in the red box in the picture above (first hold down `Fn` and then hold down win)

## 3 Volume adjustment and playback control

My only dissatisfaction with Amilo's keyboard is that the function keys are not labeled, so that every time I press `Fn', it is a matter of life or death, and Amilo's instructions are also very sloppy:

![image-20250218150253087](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250218150253087.png)

Which are the play buttons on this?  

Here, the author has rewritten the above table for daily use:

| Key combination | Result |
| ---------- | ---------- |
| `Fn`+`F7` | Previous song |
| `Fn`+`F8` | Pause/Play |
| `Fn`+`F9` | Next song |
| `Fn`+`F10` | Mute |
| `Fn`+`F11` | Volume down |
| `Fn`+`F12` | Volume increase |

## 4 Instructions for using function keys

![IMG_20250218_150757](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/IMG_20250218_150757-1739862454781-2.jpg)

The functions of the four buttons in the upper right corner of the keyboard from left to right are:

| Key location | Function |
| ----------------------- | --------------- |
| `The first function key from the left in the upper right corner` | Open the player |
| `The second function key from the left in the upper right corner` | Open the browser |
| `The third function key from the left in the upper right corner` | Open the calculator |
| `The 4th function key from the left in the upper right corner` | Open the file manager |
