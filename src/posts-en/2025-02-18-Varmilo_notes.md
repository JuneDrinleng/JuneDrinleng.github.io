## 0 Background
The author purchased an Amilo Hua Dan Nang Red-axis Mechanical Keyboard in 2024, being attracted by its excellent appearance and the familiar Red-axis switches. However, during nearly one year of usage, the author encountered many issues, including but not limited to the Win key suddenly becoming unresponsive (the Win key on the laptop still works), the entire mechanical keyboard being locked, not knowing where to play/pause/switch songs, adjust volume, mirror audio, and what the four keys in the top-right corner correspond to. After searching the internet, some issues remained unsolved, so the author consulted the customer service today to resolve them.  
To avoid forgetting and to facilitate users, the author wrote this notes to record the keyboard's usage.

## 1 Keyboard Lock
Unlike Cherry's Red-axis MX250 keyboard, the volume key adjustment on Amilo's keyboard is not on `Fn`+`F2`. Therefore, when the author tried `Fn`+`F2` multiple times, the mechanical keyboard locked directly and became unusable.  

When the keyboard is locked, the first of the three indicator lights in the top-left corner will turn off. To resolve this issue, simply press `Fn`+`Del`.  

## 2 Win Key Failure
As a heavy user of the Win key, its failure caused significant inconvenience (who doesn't have apps on their desktop, with all apps collected in the Win key menu). The method to trigger the Win key failure (or lock the Win key) is simple: press `Fn` and the Win key on its right side:  

![IMG_20250218_150028](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/IMG_20250218_150028.jpg)

Generally, triggering this is very accidental, so to unlock, simply press the two keys in the red box in the image above (press `Fn` first, then press Win).  

## 3 Volume Adjustment and Playback Control
The only dissatisfaction the author has with the Amilo Hua Dan Nang keyboard is that these function keys are not labeled, making each press of `Fn` unpredictable. The Amilo manual is also very hasty:  

![image-20250218150253087](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250218150253087.png)

Which of these are the play keys?  

Here, the author has recompiled the table above for daily use:  

| Key Combination | Result      |
|------------------|-------------|
| `Fn`+`F7`        | Previous Track |
| `Fn`+`F8`        | Pause/Play   |
| `Fn`+`F9`        | Next Track   |
| `Fn`+`F10`       | Mute         |
| `Fn`+`F11`       | Volume Down  |
| `Fn`+`F12`       | Volume Up    |

## 4 Function Key Usage Guide

![IMG_20250218_150757](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/IMG_20250218_150757-1739862454781-2.jpg)

The four buttons on the top-right corner of the keyboard, from left to right, have the following functions:  

| Key Position                | Function           |
|-----------------------------|--------------------|
| `Leftmost Function Key on Top-Right` | Open Media Player |
| `Second Function Key on Top-Right`  | Open Browser      |
| `Third Function Key on Top-Right`   | Open Calculator   |
| `Fourth Function Key on Top-Right`  | Open File Manager |
