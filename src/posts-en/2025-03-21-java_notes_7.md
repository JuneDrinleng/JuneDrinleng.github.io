## 1 Character type
A single character is a special type: char
~~~java
char c='a';
char d='Chinese';
~~~

A single Chinese character is also a character

## 4 String variable

The content enclosed in double quotes is generally a string. The variable used for strings is `String`. It should be noted that it is a package type. The string variable is the manager of the string rather than the owner. Variables that create strings must also be created with `new`

~~~java
String s = new String ("a string");
~~~

You can also initialize it directly:

~~~java
String s="hello string";
~~~

String links can be achieved using `+`. If the other side of `+` is not a string, Java will automatically convert the content that is not a string into a string.

~~~java
String s="hello";
System.out.println(s+20+25)
~~~

What you get is hello2025 instead of hello45

For input, there are some differences:

~~~java

Enter this is an apple
s only has this

s=in.nextln(); you can read the contents of the entire line
~~~

## 5 String operations

String ratio size:

~~~java
s1.compareTo(s2);
"abcd".compareTo(s2);
~~~

Compare the string content of two string variables s1 and s2, which one is greater?

To calculate the length of a string, use length. It should be noted that `length()` is a right parenthesis.

~~~
String s="hello";
s.length() is 5
~~~

To traverse a string you can use:

~~~
for (int i=0; i<s1.length();i++){
	System.out.println(s1.charAt(i));
}
~~~

where `s1` is a string variable
