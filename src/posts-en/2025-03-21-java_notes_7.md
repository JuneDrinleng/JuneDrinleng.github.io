## 1 Character Type
A single character is a special type: char
~~~java
char c='a';
char d='汉';
~~~

A single Chinese character is also a character

Incrementing a character will result in the next character in the Unicode table. After performing addition or subtraction operations on a character, the result is a number rather than a character

![image-20250321221207558](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250321221207558.png)

You can also provide the Unicode number, and the program can output the corresponding character

## 3 Wrapper Types

![image-20250321222306555](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250321222306555.png)

Wrapper types, like primitive types, can be used to define variables. They can perform the same operations as primitive types while also offering unique functionalities of wrapper types:

![image-20250321222457272](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250321222457272.png)

Here, the `.` is an operator used to let a class or object perform certain actions

There is also a method to convert case:

~~~java
System.out.println(Character.toLowerCase('I'));
~~~

And a method to check case:

~~~java
System.out.println(Character.isLowerCase('I'));
~~~

## 4 String Variables

Content enclosed in double quotes is generally a string. The variable type for strings is `String`. Note that it is a wrapper type. A string variable manages rather than owns the string. Creating a string variable also requires using `new` to create it

~~~java
String s = new String ("a string");
~~~

It can also be initialized directly:

~~~java
String s="hello string";
~~~

String concatenation can be achieved using `+`. If the other side of `+` is not a string, Java automatically converts non-string content into a string

~~~java
String s="hello";
System.out.println(s+20+25)
~~~

The result is hello2025 rather than hello45

For input, there are some differences:

~~~java
s=in.next();
Input this is an apple
s only has this

s=in.nextLine(); can read the entire line's content
~~~

## 5 String Operations
String Comparison:

~~~java
s1.compareTo(s2);
"abcd".compareTo(s2);
~~~

Compare the string content of s1 and s2 variables

Calculate string length using length, note that `length()` uses a right parenthesis

~~~
String s="hello";
s.length() is 5
~~~

Traverse a string using:

~~~
for (int i=0; i<s1.length();i++){
	System.out.println(s1.charAt(i));
}
~~~

Where `s1` is a string variable
