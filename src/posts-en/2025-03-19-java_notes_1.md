## 1 Computer Basic Concepts

A common method for computers: enumeration method

However, enumeration methods are often slow. To accelerate computation, many other methods have been designed, such as binary search, or more complex optimized methods

Regarding program execution:

![image-20250319193546157](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250319193546157.png)

The interpreter executes the code line by line according to the code, while the compiler translates the code into machine code and runs it directly with the machine code

It is important to note that the distinction between interpretation and compilation of a language is not specific. It is merely a matter of habit that we often choose, for example, Python as an interpreter and C as a compiler.

Today, these two approaches have no significant differences.

After writing the code, it can be found that there is a warning below, which is due to Oracle introducing stricter security restrictions on native methods (such as `System.load`) in Java 17 and later versions, causing IntelliJ IDEA to trigger warnings when running Java code.

To resolve this issue, the configuration needs to be adjusted in `Run → Edit Configurations`:

![image-20250319232716051](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250319232716051.png)

Add `--enable-native-access=ALL-UNNAMED` in the "Add VM options" section.

After running, there will be no warnings:
![image-20250319232813512](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250319232813512.png)

> Tip: The content of `public static void` can be directly added using the `psvm` shortcut.

It is important to note that each line should end with an English semicolon (like in C), otherwise an error will occur.

## 4 Input Content
~~~
import java.util.Scanner;

public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello");
        Scanner in = new Scanner(System.in);
        System.out.println("echo:" + in.nextLine());
    }
}
~~~

The `import` statement is automatically generated

If you want to add some fixed strings, similar to Python, you can use the `+` operator:

~~~
import java.util.Scanner;

public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello");
        Scanner in = new Scanner(System.in);
        System.out.println("echo:" + in.nextLine());
    }
}
~~~
