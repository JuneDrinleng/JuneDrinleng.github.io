## 1 Basic Computer Concepts

A common method of computers: enumeration method

However, enumeration methods are often very slow. In order to speed up calculations, many other methods are often designed, such as the binary method, etc., or more complex optimization methods.

Regarding the execution of the program:

![image-20250319193546157](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250319193546157.png)

The interpreter executes the code line by line each time, while the compiler compiles the code into machine code and then runs it directly with the machine code.

It should be noted that there is no specific distinction between language interpretation and compilation. It is just based on habits that for example, Python is the interpreter and C uses the compiler.

Today, there is no obvious difference between the two approaches.

After writing, you can find that there is a warning below. This is because Oracle has introduced stricter security restrictions on Java's native methods (such as `System.load`) since **Java 17** and later versions, causing IDEA to trigger a warning when running Java code.

In order to solve this problem, you need to configure it in `Run → Edit Configurations`:

## 4 Input content

~~~
import java.util.Scanner;

public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello");
        Scanner in = new Scanner(System.in);
        System.out.println(in.nextLine());
    }
}
~~~

The import here is automatic

If you want to add some fixed strings, similar to python, you can use the plus sign:

~~~
import java.util.Scanner;

public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello");
        Scanner in = new Scanner(System.in);
        System.out.println("echo:"+in.nextLine());
    }
}
															
~~~
