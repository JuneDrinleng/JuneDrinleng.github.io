## Step 3: Centering All Text Elements
To adjust the text and title formatting, we need to use CSS for styling. Typically, the CSS file is named `style.css` and stored in the same directory as the HTML file. For easier manipulation of individual elements, we can add classes to tags and modify their styles in the CSS file. Note that after using CSS, we need to import the CSS file in the HTML: `<link rel="stylesheet" href="./styles.css">`

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Catbook</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>

  <body>
    <h1>Catbook</h1>
    <img src="./cat.png" alt="A cute cat" />
    <h1 class="u-textCenter">Buka Buka</h1>
    <h4>About Me</h4>
    <p>I am more of a turtle person but I'm just trying to fit in and get a catbook.</p>
    <h4>My Favorite Type of Cat</h4>
    <p>I actually prefer turtles.</p>
  </body>
</html>
```

Then, create a class in `styles.css`:

```css
.u-textCenter {
  text-align: center;
}
```

Use `text-align` to center the corresponding class.

## Step 5: Add Navbar
Our goal's topmost catbook title has a banner, which is called navbar. We can create a `nav` tag to include the catbook title within, and finally set the navbar's style in CSS.

~~~html
<!DOCTYPE html>
<html>
  <head>
    <title>Catbook</title>
    <link rel="stylesheet" href="./styles.css" />
    <link rel="preconnect" href="https://fonts.googleapis.com" />
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
    <link
      href="https://fonts.googleapis.com/css2?family=Poetsen+One&display=swap"
      rel="stylesheet"
    />
  </head>

  <body>
    <nav class="navContainer">
      <h1 class="navTitle">Catbook</h1>
    </nav>
    <img src="./cat.png" alt="A cute cat" />
    <h1 class="u-textCenter">Buka Buka</h1>
    <section class="u-textCenter">
      <h4>About Me</h4>
      <p>I am more of a turtle person but I'm just trying to fit in and get a catbook.</p>
      <h4>My Favorite Type of Cat</h4>
      <p>I actually prefer turtles.</p>
    </section>
  </body>
</html>

~~~

We only need to adjust the font of the `navTitle` object. This can be achieved by setting styles in the CSS file.

~~~css
.u-textCenter {
  text-align: center;
}
body {
  font-family: "Poetsen One", sans-serif;
  font-weight: 400;
  font-style: normal;
}
.navTitle {
  font-family: "Poetsen One", sans-serif;
  font-weight: 400;
  margin: 0;
  font-size: 2rem;
  text-align: center;
  color: var(--primary);
}
:root {
  --primary: #396dff;
  --grey: #f7f7f7;
  --white: #ffffff;
}

~~~

Here, we set some common colors in the `:root` for easy access by other elements. The `margin` is used to set the browser's margin to 0.

Here, the image's `class` is named `avatar`, and we adjust the properties of the `avatar` object in `styles.css`:

To make the image circular, we need to use `border-radius`. Generally, we can also set several breakpoints in the root, such as 16px, 8px, 4px, 2px, and 32px.

~~~css
.u-textCenter {
  text-align: center;
}
body {
  font-family: "Poetsen One", sans-serif;
  font-weight: 400;
  font-style: normal;
  margin: 0 0;
}
.navTitle {
  font-family: "Poetsen One", sans-serif;
  font-weight: 400;
  margin: 0;
  font-size: 2rem;
  text-align: center;
  color: var(--white);
}
.navContainer {
  background-color: var(--primary);
  padding: 8px 16px;
}
:root {
  --primary: #396dff;
  --grey: #f7f7f7;
  --white: #ffffff;
  --m: 16px;
  --s: 8px;
  --xs: 4px;
  --l: 24px;
  --xl: 32px;
~~~

Here, the ratio is set to 50%, resulting in:

We can see it's an ellipse because we haven't set the width and height to be equal, and the avatar is also too large. We can scale it using `width` and `height`, and set `aspect-ratio: 1/1;` to make the width and height equal. We can also set the margin and padding properly:

~~~css
.avatar {

  border-radius: 50%;
~~~

## step8 Process Text Columns
To achieve the layout, a one-dimensional layout tool called flex is used. The section to be laid out is as follows:

~~~html
<section class="u-textCenter">
      <h4>About Me</h4>
      <p>I am more of a turtle person but I'm just trying to fit in and get a catbook.</p>
      <h4>My Favorite Type of Cat</h4>
      <p>I actually prefer turtles.</p>
</section>
~~~

We can place the entire section inside a flex container, with each subsection being a flex child container. One child container holds "About Me" and another holds "My Favorite Type of Cat":

~~~html
<div class="u-flex">
    <section class="u-textCenter">
    	<h4>About Me</h4>
        <p>I am more of a turtle person but I'm just trying to fit in and get a catbook.</p>
    </section>
    <section class="u-textCenter">
    	<h4>My Favorite Type of Cat</h4>
        <p>I actually prefer turtles.</p>
    </section>    
</div>
~~~

The parent container needs to enable flex, corresponding to the `display:flex;` in the CSS file.

Since the "About Me" child container (child container 1) appears before the "My Favorite Type of Cat" child container (child container 2), we want the layout to have child container 2 on the left and child container 1 on the right. The default flex layout `row` arranges children from left to right: [child container 1] [child container 2]. Therefore, we reverse the layout using `row-reverse`:

~~~
.u-flex {
  display: flex;
  flex-direction: row-reverse;
}
~~~

After the above adjustments, the result is as follows:

![image-20250528173337659](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250528173337659.png)

This is because we only set the sorting of the flex container without specifying the formatting and layout of the child containers.

We need to set the starting position of the child containers to 0 (indicating fixed positioning), and ensure both child containers have the same size. To achieve this, we can apply the same class `.subContainer` to both child containers, constraining them under identical rules.

~~~css
/* Your Code Here! */

.u-textCenter {
  text-align: center;
}
body {
  font-family: "Poetsen One", sans-serif;
  font-weight: 400;
  font-style: normal;
  margin: 0 0;
}
.navTitle {
  font-family: "Poetsen One", sans-serif;
  font-weight: 400;
  margin: 0;
  font-size: 2rem;
  text-align: center;
  color: var(--white);
}
.navContainer {
  background-color: var(--primary);
  padding: 8px 16px;
}
:root {
  --primary: #396dff;
  --grey: #f7f7f7;
  --white: #ffffff;
  --m: 16px;
  --s: 8px;
  --xs: 4px;
  --l: 24px;
  --xl: 32px;
}
.avatar {
  width: 10%;
  height: 10%;
  border-radius: 50%;
  aspect-ratio: 1/1;
  object-fit: cover;
  display: block;
  margin-left: auto;
  margin-right: auto;
  padding: 8px 16px;
}
.u-flex {
  display: flex;
  flex-direction: row-reverse;
}
.subContainer {
  flex-grow: 1;
  flex-basis: 0;
}

~~~

~~~html
<!DOCTYPE html>
<html>

        <h4>About Me</h4>
        <p>I am more of a turtle person but I'm just trying to fit in and get a catbook.</p>
      </section>
      <section class="u-textCenter subContainer">
        <h4>My Favorite Type of Cat</h4>

~~~

If we want to set the size ratio of the two child containers to be inconsistent, we need to define separate classes for each. For example, class1 has `flex-grow: 1` and class2 has `flex-grow: 2`. The absolute numerical values of `flex-grow` are not significant; what matters is the proportion relative to other child containers within the parent container, which determines their size.
