## step3 Center these introduction fonts

In order to adjust the words of these introductions and titles, we need to use css for beautification. Generally, the css file is named style.css and is stored in a directory parallel to the html. In order to more conveniently control each element, we can beautify it by adding a class to the tag and then modifying and adjusting the format of the class in css. It should be noted that after using css, you need to introduce the css file into the html: `<link rel="stylesheet" herf="./styles.css">`

~~~html
<!DOCTYPE html>
<html>
  <head>
    <title>Catbook</title>
    <link rel="stylesheet" href="./styles.css" />
  </head><body>
    <h1>Catbook</h1>
    <img src="./cat.png" alt="A cute cat" />
    <h1 class="u-textCenter">Buka Buka</h1>
    <h4>About Me</h4>
    <p>I am more of a turtle person but I'm just trying to fit in and get a catbook.</p>
    <h4>My Favorite Type of Cat</h4>
    <p>I actually prefer turtles.</p>
  </body>
</html>

~~~

Then create a class in styles.css:

~~~css
.u-textCenter {
  text-align: center;
}

~~~

Use text-align to center corresponding classes

  </body>

~~~

Then we only need to adjust the font of the `navTitle` object, which can be set in the css file.~~~css

## step8 Process text columns

In order to implement layout, the one-dimensional layout tool flex is used here. What we need to layout is the following parts

~~~html
<section class="u-textCenter">
      <h4>About Me</h4>
      <p>I am more of a turtle person but I'm just trying to fit in and get a catbook.</p>
      <h4>My Favorite Type of Cat</h4>
      <p>I actually prefer turtles.</p>
</section>
~~~We can put the entire part in a flex container. Each small part is a sub-container of flex. One sub-container stores about me and one sub-container stores My Favorite Type of Cat.

~~~html
<div class="u-flex">
    <section class="u-textCenter">
    	<h4>About Me</h4>
        <p>I am more of a turtle person but I'm just trying to fit in and get a catbook.</p>
    </section>
    <section class="u-textCenter">
    	<h4>My Favorite Type of Cat</h4>
        <p>I actually prefer turtles.</p>
    </section>
</div>
~~~

The parent container needs to enable flex, which corresponds to `display:flex;` in the css file.And since our about subcontainer (subcontainer 1) is before the favorite subcontainer (subcontainer 2), we want the layout to be such that subcontainer 2 is on the left and subcontainer 1 is on the right, and the default flex layout `row` is laid out from left to right: [Subcontainer 1] [Subcontainer 2], so our layout must be reversed `row-reverse`

~~~
.u-flex {
  display: flex;
  flex-direction: row-reverse;
}
~~~

After the above, the following results are obtained:

![image-20250528173337659](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250528173337659.png)

This is because we only set the sorting within the flex container and did not set the format and layout of the two sub-containers.

We need to set the starting position of the subcontainer to 0 (indicating freeze frame), and the size of the subcontainer must be consistent and the same, then we only need to set `.subContainer`, then both subcontainers comply with the constraints of the same class~~~css
