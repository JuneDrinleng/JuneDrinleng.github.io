## 1 Variables

import java.util.Scanner;

public class Hello {
    public static void main(String[] args) {
//        System.out.println("Hello");

        System.out.println("echo:"+in.nextLine());

## 2 Assignment

In computers, `=` represents the assignment operation rather than mathematical equality:

a=6 actually means: a <-- 6

This implies that assignment in computers does not follow the commutative law

If a variable is used before assignment, it will result in an error:

![image-20250320104252848](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250320104252848.png)

## 5 Type Conversion

![image-20250320201644901](https://raw.githubusercontent.com/JuneDrinleng/JuneDrinleng.github.io/main/assets/img/image-20250320201644901.png)

The calculation results here often have many decimal places, but we usually use values like 170cm. Therefore, type conversion is needed:

~~~
import java.util.Scanner;

public class inch {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        double inch, foot;
        System.out.print("Enter feet: ");
        foot = in.nextDouble();
        System.out.print("Enter inches: ");
        inch = in.nextDouble();
        System.out.println((int)(100 * 0.3046 * (foot + inch / 12)) + "cm");
    }
}

~~~

You can use `(int)(float value)` to convert a floating-point number to an integer. This conversion generates a new number and does not affect the original value.
